# 실행 컨텍스트의 생성과 식별자 검색 과정
```typescript jsx
var x = 1;
const y = 2;

function foo(a) {
	var x = 3;
	const y = 4;
	
	function bar(b) {
		const z = 5;
		console.log(a + b + x + y + z)
  }
	
	bar(10)
}

foo(20) // 42
```
## 1) 전역 객체 생성
- 전역 객체는 전역 코드가 평가되기 이전에 생성된다
- 전역 객체도 Object.prototype 을 상속 받는다.
  - 즉, 전역 객체도 프로토타입 체인의 일원
